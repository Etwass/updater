# Updater

Updater — это открытая утилита под лицензией **GPLv3**, предназначенная для автоматического обновления данных на компьютере пользователя.  
Программа по расписанию подключается к FTP‑серверу, скачивает новые файлы и заменяет ими старые.

## Возможности
- Поддержка FTP(S)
- Проверка целостности загруженных файлов
- Автоматическая замена данных
- Логирование действий
- Гибкая конфигурация расписания
- Полностью открытый исходный код

## Лицензия
