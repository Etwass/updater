## 🇬🇧 English

**Updater** is an open‑source utility licensed under **GPLv3**, designed to automatically update data on the user's computer.  
The program connects to an FTP server on a schedule, downloads new files, and replaces the old ones.

### Features
- FTP(S) support
- File integrity verification
- Automatic data replacement
- Action logging
- Flexible scheduling configuration
- Fully open source
